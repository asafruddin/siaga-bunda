# UX flow

Respondent: splash → onboarding → three registration steps → dashboard → pretest intro → 10 questions → constrained video player → seven-day waiting state → posttest intro (from dashboard) → 10 questions → completion and next-video unlock. Locked cards always explain the prerequisite; loading, empty, and API-error states remain in context.

Researcher: onboarding → protected login → mobile summary cards → respondent search/filter/pagination or video/result summaries → respondent detail timeline. Export offers dataset type, date, video, and status filters, then opens the native share sheet.

The visual system uses large controls, high-contrast body text, soft pink/purple medical colors, friendly Indonesian copy, and one primary action per step.
