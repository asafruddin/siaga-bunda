- (decision) Exports are template-bound asynchronous jobs generated by the worker into KMS-encrypted S3 controlled destinations; mobile can request and observe but never downloads through an unrestricted share sheet.
- (decision) Source Records are insert-only; corrections append linked records. Audit events use a dedicated append role, update/delete-denying triggers, chained digests, and periodic digest anchoring to S3 Object Lock.
- (decision) Version and atomically activate Study configuration, Consent, content, assessments, scoring, urgent-care guidance, retention, and export templates; activation blocks unless approvals and consistency checks pass.
- (decision) Mobile keeps server cache in memory only, tokens and small assessment-choice drafts in SecureStore, registration/Consent drafts in memory, and no answer keys or privileged credentials; foreground always reconciles backend state.
