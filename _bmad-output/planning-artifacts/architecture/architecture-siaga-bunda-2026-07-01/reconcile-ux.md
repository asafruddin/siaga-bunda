# Architecture Reconciliation — UX Spines

## Result

- Expo/React Native with Material-compatible primitives supports the Android-only visual and behavioral contract.
- Role-separated route groups, in-memory server cache, scoped SecureStore state, screen-capture controls, and foreground reconciliation preserve the spines’ privacy and stale-state rules.
- Exact Delay display derives from backend instants; boundary refresh never trusts device time.
- Loading, offline, interrupted playback, submission uncertainty, notification denial, re-consent, suspension, withdrawal, and export states all have backend status or idempotent recovery semantics.
- Urgent-care Guidance is versioned by Study configuration and a baseline can be bundled for first-launch offline access.

No UX decision requires a conflicting architecture exception.
