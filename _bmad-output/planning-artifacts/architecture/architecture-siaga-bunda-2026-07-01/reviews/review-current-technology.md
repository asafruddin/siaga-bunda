# Current Technology Verification

## Verdict

Verified against primary project or vendor sources on 2026-07-01.

- Expo SDK 56.0.0 is the current stable Expo SDK and targets React Native 0.85, React 19.2.3, Node 22.13 minimum, and Android API 36.
- React Native 0.85 is supported through the Expo 56 paved path even though React Native 0.86 is independently active.
- Node.js 24.18.0 is LTS.
- NestJS 11.1.24 is the latest listed framework release.
- PostgreSQL 18 is the current stable documentation line; PostgreSQL 19 is beta and was not selected.
- Prisma ORM 7.7.0 is the current production/LTS line in the official changelog/roadmap.
- pnpm 11.4.0 and Turborepo 2.9.15 are current releases.
- TypeScript 5.9 and Zod 4 are current stable lines.
- AWS CDK 2.257.0 and AWS SDK for JavaScript 3.1055.0 are current releases.
- Maestro CLI 2.4.0 is current and officially supports React Native/Android through the accessibility layer.
- Cognito user pools and identity endpoints exist in `ap-southeast-3`; Cognito officially supports SMS OTP passwordless and TOTP MFA. Separate pools avoid their incompatibility in one user’s flow.

No stale or nonexistent technology was found.
