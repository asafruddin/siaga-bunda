# SiAGA Bunda mobile

Android-first Expo SDK 56 scaffold for the Participant and Researcher experiences. The current public screen is intentionally non-functional; protected routes always deny access until the authentication story supplies a verified session.

Run `pnpm --filter @siaga-bunda/mobile start` for a development client. EAS project linking, signing credentials, Play registration, production services, clinical content, and production OTA updates are not configured.
